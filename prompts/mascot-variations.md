# ときめきラボ マスコットバリエーション定義書

## コンセプト

ベースは同じ丸い可愛い生物（たまごっち/すみっコぐらし風）。
差分要素: 体色、髪型/頭の飾り、表情、目の形、体のサイズ（大小）、持ち物/アクセサリ1つ。
同じ診断内のキャラたちは「家族」のように似ているが個性がある。

### サブタイプの濃淡ルール

各ベースタイプから8つのサブタイプが派生。以下の3軸で色の濃淡を調整:
- **果断(A) vs 慎重(O)**: 果断=やや明るく鮮やか / 慎重=やや暗く深い
- **熱意(H) vs クール(C)**: 熱意=暖色方向にシフト / クール=寒色方向にシフト
- **安定(S) vs 反応(R)**: 安定=彩度を抑えた落ち着き / 反応=彩度を上げた鮮やか

例: INTJ（#4C1D95 ダークパープル）の場合
- INTJ-A-H-S: #6B3FA0（明るく暖かく落ち着き）
- INTJ-A-H-R: #7B2FAA（明るく暖かく鮮やか）
- INTJ-A-C-S: #3D1F90（明るく冷たく落ち着き）
- INTJ-A-C-R: #4D0F9A（明るく冷たく鮮やか）
- INTJ-O-H-S: #5C2D80（暗く暖かく落ち着き）
- INTJ-O-H-R: #6C1D8A（暗く暖かく鮮やか）
- INTJ-O-C-S: #2C1D70（暗く冷たく落ち着き）
- INTJ-O-C-R: #3C0D7A（暗く冷たく鮮やか）

## 差分設計の共通ルール（まとめ）

### 体色
- タイプの color をベースに、パステルトーンに調整
- 暖色系タイプ（情熱的）→ ピンク/赤/オレンジ系
- 冷色系タイプ（冷静）→ 青/紫/緑系
- 中間系タイプ → 黄/ラベンダー/ミント系

### 髪型/頭飾り
- リーダー系 → ツンと立った髪/冠
- 穏やか系 → ふわふわ丸い髪
- 知的系 → メガネ/本の形の髪飾り
- 野生系 → ボサボサ/尖った髪
- 芸術系 → カラフル/ウェーブ
- 戦闘系 → はちまき/角
- 癒し系 → 花冠/リボン

### 表情
- 自信家 → キリッとした顔、片目ウインク
- 穏やか → にこにこ、目が三日月
- クール → 無表情～微笑、半開き目
- 情熱的 → キラキラ目、大きく開いた口
- ミステリアス → 片目隠れ、薄笑い
- お調子者 → べー（舌出し）、にやり
- 繊細 → うるうる目、小さい口

### 目の形
- 丸目 → 明るい/純粋系
- たれ目 → 優しい/癒し系
- つり目 → クール/知的系
- 半開き → マイペース/ミステリアス系
- キラキラ目 → 情熱/夢見る系
- 糸目 → 策略家/笑顔の裏ボス系

### サイズ
- S（小さい）→ 控えめ/繊細/子供っぽいタイプ
- M（標準）→ バランス型
- L（大きい）→ リーダー/守護/パワー系
